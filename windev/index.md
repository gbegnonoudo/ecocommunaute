# Code WinDev — EcoCommunautés

Ce dossier regroupe le code WLanguage et les notes techniques à recopier ou adapter dans le projet WinDev 2025.

## Organisation

```text
windev/
├── constantes-variables/
├── procedures-globales/
├── fenetres/
└── regles/
```

## Convention

- Les fichiers `.wl` contiennent du WLanguage.
- Les fichiers `.md` expliquent le rôle des fenêtres, champs et procédures.
- Les procédures validées ne doivent pas être renommées sans décision explicite.
