# Workflow Global — EcoCommunautés

## 1. Connexion

- Authentification via Groupware Utilisateur WinDev.
- Chargement du miroir applicatif UTILISATEUR.
- Détermination du groupe : ADMIN, PROVINCIAL ou COMMUNAUTAIRE.

## 2. Paramétrage

- Gestion des communautés.
- Gestion des exercices.
- Génération des périodes comptables.
- Paramétrage des taux de change.
- Paramétrage des comptes de trésorerie.

## 3. Saisie comptable

- Création des opérations.
- Ajout des lignes débit/crédit.
- Contrôle de l'équilibre comptable.
- Association des pièces justificatives.
- Association éventuelle d'une ressource.

## 4. Contrôle communautaire

Le communautaire :

- vérifie les opérations ;
- corrige les anomalies ;
- soumet la période.

Statuts possibles :

- OUVERTE
- SOUMISE
- A_CORRIGER
- VALIDEE_PROVINCE
- CLOTUREE

## 5. Contrôle provincial

Le provincial :

- contrôle les comptes ;
- demande correction ;
- valide la période ;
- clôture la période.

## 6. Rapports

- Génération des rapports trimestriels.
- Génération du rapport annuel final.
- Conversion des montants en devise d'origine, EUR et FCFA.
- Export vers les modèles Excel officiels.
