# Manuel d'utilisation — EcoCommunautés

## 1. Connexion

L'utilisateur se connecte avec son compte prévu dans le Groupware Utilisateur WinDev.

Selon son groupe, il accède aux fonctions autorisées :

- ADMIN
- PROVINCIAL
- COMMUNAUTAIRE

## 2. Tableau de bord

Le tableau de bord donne accès aux fonctions principales.

Pour un communautaire, les boutons importants doivent être accessibles ici, notamment :

- opérations ;
- périodes comptables ;
- rapport trimestriel ;
- rapport annuel final.

## 3. Saisie des opérations

L'utilisateur saisit une opération avec :

- la date ;
- le libellé ;
- la référence de pièce ;
- le compte de trésorerie ;
- les lignes comptables.

L'opération doit être équilibrée :

```text
Total débit = Total crédit
```

## 4. Soumission d'une période

Le communautaire soumet une période lorsque toutes les opérations sont correctes.

La période passe alors au statut :

```text
SOUMISE
```

## 5. Contrôle provincial

Le provincial peut :

- retourner la période en correction ;
- valider la période ;
- clôturer la période.

## 6. Rapports

Les rapports sont générés à partir des opérations validées.

Les montants officiels sont disponibles en :

- FCFA
- EUR

## 7. Bonnes pratiques

- Ne pas saisir une opération sans pièce justificative lorsque la pièce existe.
- Vérifier les totaux avant d'enregistrer.
- Corriger les anomalies avant de soumettre une période.
- Ne pas attendre la fin de l'année pour saisir toutes les opérations.
